Non-Circular Dialogue Arrows

This version changes the arrow stroke geometry itself. The paths are speech-bubble edges, not circular loop arcs.

1. Rounded Bubble Arrows

The arrow lines form the upper-left edge and lower-right edge of a speech bubble. The loop idea comes from opposing arrow directions, not a circle.

Recommended: best fit for your correction: arrow line shape is visibly dialogue-shaped.

2. Two Bubble Edge Arrows

Two opposing message edges: top arrow is one message, bottom arrow is the reply. Very direct and less circular.

Tradeoff: extremely simple, but less like one continuous Converloop mark.

3. Speech Tail Turn

The return arrow line makes a sharper speech-tail turn. It is the most explicit bubble shape, but slightly less elegant.

Tradeoff: strong dialogue cue, less smooth.

What changed

The main stroke path is no longer circular. It uses vertical sides, horizontal message edges, rounded corners, and a speech-tail turn. The arrowheads remain, but the line shape now carries the dialogue concept.
